A location puck is used to mark a user's location on the map.

## When to use

Use the location puck to represent a user's current location on a map. Do not use this component if the user's current location is not available. Do not use this component to represent anything other than a user's current location.

## Guidelines

The location puck comes in two types: `consumer` and `earner`.

The `consumer` location puck should generally be used in map views in consumer applications, such as Rides or Eats apps.

The `earner` location puck should generally be used in navigation-based applications, such as the Earner app. The `earner` puck can be used for overview or turn-by-turn navigation use cases.
